# Basketball Trainer Pro

A lightweight, premium web application for managing basketball exercises and designing training sessions.

## Features

-   **Exercise Library**: Create and manage a database of basketball drills.
-   **Tactical Whiteboard**: Draw custom plays and drills on a half-court canvas.
-   **Training Builder**: Assemble training sessions by selecting exercises from your library.
-   **Cloud Sync**: Data is stored securely in Firebase Firestore, accessible from any device.
-   **Responsive Design**: Works on desktop, tablets, and mobile devices.

## Tech Stack

-   **Frontend**: HTML5, CSS3, JavaScript (Vanilla).
-   **Backend**: Firebase (Firestore).
-   **Icons**: FontAwesome.
-   **Fonts**: Google Fonts (Outfit).

## Usage

1.  Open `index.html` in any modern web browser.
2.  Start adding exercises and creating trainings immediately.

## Deployment

This project is a static web application. It can be hosted on GitHub Pages, Vercel, Netlify, or any static file host.

See `DEPLOYMENT_GUIDE.md` for instructions on how to publish it on GitHub.
